Two kinds of rolls: Skill Tests and Effect Magnitudes.

-   For a skill sest, state an action and the skill being used.

    -   GM has a **standard difficulty** threshold or an **opposition difficulty** from a character opposing the action.

    -   GM may have difficulty modifiers for setting and situation, also.

-   For an effect magnitude, a skill test's result will dictate the number of dice.

1. **Gather Dice**, *n*\ **D**\ +\ *m*.  One die must be an Odd-colored "Wild Die".

    For a Skill Test, look at attributes, skills, special abilities in use.
    Don't forget advantages and disadvantages.

    For an Effect Magnitude, look at the skill result, weapon, skill, special ability for the number of dice.

    GM may include modifiers based on skill test results, setting, situation, or opposition.

3.  **Spend a Fate Point** to Roll double the dice.

4.  **Roll**.

    ..  csv-table:: Wild Die Adjustments

        Wild Die, Result
        1, "**Failure**. Discard wild die and highest number die."
        6, "**Success**. Roll the wild die again. Accumulate values; stop when a non-6 is rolled."

5.  **Spend a Character Point** to add a die.

6.  **Result**.

    -   For a Skill Test, two kinds of results:

        -   Skill roll ≥ Difficulty: success of some kind. See `Result Points`_; this may impact the effect magnitude.

        -   Skill roll < Difficulty: failure.

    -   For Effect Magnitude this is the body points, or duration, or range, etc.
        The GM will help with Damage

Result Points
-------------

When skill roll meets or exceeds difficulty.

..  csv-table:: Result Points

    0,Success. Barely
    1-4,Solid
    5-8,Good; possibly an added benefit
    9-12,Superior; added benefits
    13-16,Spectacular
    >17,Incredible; significant bonus

Result points ÷ 2 can be applied to subsequent Related Skill actions as a + modifier.

In combat, result points ÷ 5 is bonus damage.
