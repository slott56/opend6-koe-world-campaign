OpenD6 Kingdom of the East
==========================

Welcome to a unique *OpenD6 Fantasy* world, the **Kingdom of the East**, and a campaign, **The Bandit King**.
The two principle books are the worldbook, and a campaign:

-   :external:ref:`world.world_book`.

-   :external:ref:`campaign_1`.

As background for this world book, some clones of OGL-licensed base *OpenD6* rules are also provided here.
These are scraped from PDF's for the most part and have been lightly edited to correct OCR errors.

- :external:ref:`fantasy_rulebook`.

- :external:ref:`fantasy_locations`.

- :external:ref:`players_guide`.

- :external:ref:`gm_ref`.

- :external:ref:`magic_guide`. (This was unpublished.)

- :external:ref:`system_book`. (A few selected snippets from this book.)

See https://ogc.rpglibrary.org/index.php?title=OpenD6 for the source documents.
These are covered by an OGL, and owned by Purgatory Publishing.

To help with spell and character design, there are some software tools, you might find helpful.
The `OpenD6 Tools <https://github.com/slott56/opend6-tools>`_ project includes some Domain-Specific Languages (DSL's)
for creating characters, creatures, items, spells, and invocations.
These DSL's and tools were used to assure the definitions in these extracts are consistent, and the computations follow the rules.

A consequence of using the OpenD6 Tools is  some of the difficulty values and spell details will be
slightly different from the original material.
For more information, see https://slott56.github.io/opend6-tools/_build/html/notes/index.html.
