OpenD6 Kingdom of the East¶

Welcome to a unique OpenD6 Fantasy world, the Kingdom of the East, and a campaign, The Bandit King. The two principle books are the worldbook, and a campaign:

  • OpenD6 Fantasy Worldbook: Kingdom of the East.

  • OpenD6 Fantasy Campaign: The Bandit King.

As background for this world book, some clones of OGL-licensed base OpenD6 rules are also provided here. These are scraped from PDF’s for the most part and have been lightly edited to correct OCR errors.

  • OpenD6 Fantasy Rulebook.

  • OpenD6 Fantasy Locations.

  • OpenD6 Fantasy Player Summary.

  • OpenD6 Game Master Reference.

  • OpenD6 Magic Guidebook. (This was unpublished.)

  • OpenD6 System Book. (A few selected snippets from this book.)

See https://ogc.rpglibrary.org/index.php?title=OpenD6 for the source documents. These are covered by an OGL, and owned by Purgatory Publishing.

To help with spell and character design, there are some software tools, you might find helpful. The OpenD6 Tools project includes some Domain-Specific Languages (DSL’s) for creating characters, creatures, items, spells, and invocations. These DSL’s and tools were used to assure the definitions in these extracts are consistent, and the computations follow the rules.

A consequence of using the OpenD6 Tools is some of the difficulty values and spell details will be slightly different from the original material. For more information, see https://slott56.github.io/opend6-tools/_build/html/notes/index.html.
