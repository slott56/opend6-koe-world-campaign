Game Basics
================

**Dice**. See :ref:`mechanics`.

**Skills**. GM picks difficulty ± modifiers.
Player picks skill ± modifiers.
See :ref:`mechanics`.

**Time**.

1 round = 5 sec

1 minute = 12 rounds

See also, combat initiative in :external:ref:`fantasy.combat`.

**Difficulties**. See :ref:`mechanics`.

**Success**. See :ref:`mechanics`.

Genre Conversion
----------------

Some rules are written using only Space or Adventure attributes, and need to be translated to Fantasy.

..  csv-table:: Genre Conversion
    :header: D6 Space, D6 Adventure, D6 Fantasy

    Agility, Reflexes, Agility
    Knowledge, Knowledge, Intellect
    Mechanical, Coordination, Coordination
    Perception, Presence, Charisma
    Strength, Physique, Physique
    Technical. Perception, Acumen
