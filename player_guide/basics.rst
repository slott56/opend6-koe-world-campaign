Character Basics
================

Attributes
----------

Distribute 18D; most are range 1D to 5D.
Extranormal can be 0D.

1D = 3 pips; +1 is ⅓D, +2 is ⅔D.

:Agility: Balance, limberness, quickness, and full-body motor abilities.
:Coordination: Hand-eye coordination and fine motor abilities.
:Physique: Physical power, and resistance to damage.
:Charisma: Emotional strength, physical attractiveness, and personality.
:Intellect: Memory and ability to learn.
:Acumen: Mental quickness, creativity, and attention to detail
:Extranormal:
    **Magic**: alteration, apportation, divination, conjuration

    **Miracles**: divination, favor, strife

Move
----

10 meters per 5 sec. round

Body Points
-----------

Roll stamina skill (Physique attribute) + 20

Strength Damage
---------------

round(Physique attribute or *lifting* skill D-level ÷ 2) D

Funds and Credits
-----------------

Base 3D

\+ These Modifiers

..  csv-table:: Funding Modifiers

    1D in Presence,-1D
    1D in Knowledge,-1D
    >= 4D in Presence,+1D
    >= 4D in Knowledge,+1D
    >= 8D in *business* skill && highest specialization,+1D

\+ any Character Points spent

Character Points
----------------

Start with 5. Spend these to add a die to a roll.

Fate Points
-----------

Start with 1. Spend these to double the dice before rolling.
